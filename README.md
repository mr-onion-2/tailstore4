# Installation
## 1. Clone the repository
```
git clone https://github.com/spacemadev/tailstore4.git
cd tailstore4
```

## 2. Install dependencies
```
npm install
```

## 3. Start the development server
```
npm run dev
```

## 4. Build for production
```
npm run build
```
